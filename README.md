# Dna-motif-search
This script was written as part of my independent bioinformatics learning.
This project explores basic bioinformatics algorithms for DNA motif discovery. Motifs are short recurring patterns in DNA sequences that may have biological significance such as binding sites or regulatory elements

##Purpose
The purpose of this project is to practice Python programming in a bioinformatics context and to understand how computational methods can be used to search for meaningful patterns in DNA sequences.


## Algorithms included

- Profile-most probable k-mer search
- clumps finding problem in DNA
- randomized motif search(Because Randomized Motif Search starts from randomly selected k-mers, the output may differ between runs. For this reason, this example does not include a fixed expected output file)
- gibbssampler(because this algorithm also includes randomized motif search, the output may differ between runs, for this reason this example does not include a fixed expected output file)
